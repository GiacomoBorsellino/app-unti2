## Installazione

1. Clona il repository

   ```sh
   git clone https://github.com/GiacomoBorsellino/app-unti2
   ```

2. Entra nella cartella

   ```sh
   Cd app-unti
   ```

3. Installa NPM

   ```sh
   npm install npm@latest -g
   npm install 
   ```

4. Avvia il progetto usando l'estensione live server
